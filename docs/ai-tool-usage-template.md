## 使い方（簡単）
1. このファイルをコピーして docs/<tool-name>-usage.md のようなファイル名で保存してください（例: docs/chatgpt-usage.md）。
2. 以下の項目に沿って実際の使用感を記入してください。
3. main ブランチに直接追加するか、新しいブランチを作るかはプロジェクトの運用に合わせてください。
